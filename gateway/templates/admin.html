Auth & sessions

How long desktop and web softphone login tokens stay valid. When a token expires, users lose outbound Caller ID list and integrations until they authorize again in the softphone settings.

Changing this affects only new logins. Existing tokens keep their original expiry — ask users to re-authorize in Callspire Settings if sessions already expired.
